MATLAB

TIPS

  1. 设置精度format long 、format short

  2. clc清空命令行，clear清除工作区，close all关闭所有图形窗口

  3. round 四舍五入

  4. 使用doc XXX，获取文档帮助

读取和载入数据

  1. 可以通过save XXX (x)来将工作区中的数据以.mat文件的形式保存在指定名文件中，也可以在其后加具体的变量名

  2. load 操作同上

向量

  1. ：操作，设置固定间隔向量，x=first:dex:end

  2. linspace，设置固定总数量向量，x=linspace(first,end.nums)

  3. 创建列向量，使用 ' 对行向量逆置即可

  4. 矩阵用space 或 ' , ' 来分割同一行的元素，用 ' ; ' 来换行，如：[1,2,3;4,5,6]，可以与创建向量的方式混用

  5. 创建随机元素：rand( ) * 10n，传入一个参数表示方阵，传入两个参数表示row x column，元素都为随机数，也可以用zeros同样的方法创建全零矩阵

    • 生成整数随机类型randi与rand不同，其第0个参数表示随机数范围，如果是单个数表示MAX，如果是 [ ] 数组表示范围，之后的参数与rand相同

  6. 通过size()函数可以得到矩阵大小(数组)，numel()函数直接返回矩阵中所有元素个数

  7. 向量与单个元素运算时为该元素与各个向量元素运算，特殊的，想对两个多维矩阵单个值之间运算需要用 ' .* '等符号

  8. 运算函数支持直接传入一个向量

数组的访问和修改

  1. 通过index访问数组时用 ( ) ，也可以用 ' : ' 来访问一系列的元素或 ' , ' 来访问矩阵某元素（此时 ' : ' 可以用来表示一整行或一整列）

  2. 获得一系列离散元素时用 [param1,param2,param3...]

  3. 存在end关键词，可以加减操作

  4. 当访问一个多维矩阵只传入一个参数时，matlab将按照列先得顺序一次访问，得到数据

  5. max、min函数会返回两个参数分别代表最大值和最大值索引，可以用 [ ] 来接受，如果只需要第二个输入，可以用~来忽略特定输出

绘制图像

  1. plot函数：用来绘制定义的向量，传入（x）、y，可添加线条样式"r-o"分别表示线条颜色、线条样式和标记点样式，可用"MarkerIndices"等标识来修改标记点

  2. 用 legend 来为图像以此添加样例

  3. 用 title 修改图像标题，xlabel、ylabel添加x、y轴标识

  4. 可以使用 xlim 函数放大所关注的区域。xlim([xmin xmax])

导入数据

  1. 使用图像化界面

  2. 当对表中不存在的元素赋值时，matlab会自动添加此元素向量

  3. 对表按某行元素排序：sortrows(elements,"XXX")

数据分析

  1. 可以对矩阵直接与某个数值进行比较，返回一个结果矩阵，代表矩阵中的每个元素与该数的关系

  2. 可以用逻辑数组来作为数组的索引，提取所有为true的元素

语法

  1. if、elseif、else 构成if语句

  2. 以 end 结束判断、循环语句

  3. 输出语句：disp("XXXX")
